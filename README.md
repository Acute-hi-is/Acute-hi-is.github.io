# ACUTE Lab Website

Website for the **ACUTE Lab** (ACoUstic and Tactile Engineering) at the University of Iceland.

**Live site:** [https://acute.hi.is](https://acute.hi.is)

## Editing Content

The website includes a local Content Management System (CMS) for editing all content without touching code.

- **[CMS User Guide (PDF)](cms/docs/USER_GUIDE.pdf)** — full guide with screenshots
- **[CMS User Guide (Markdown)](cms/docs/USER_GUIDE.md)** — same guide, viewable on GitHub

### Quick Start

```bash
cd cms
npm install    # first time only
npm run dev    # starts CMS at http://localhost:3000
```

## Built With

- [Jekyll](https://jekyllrb.com/) — static site generator
- [GitHub Pages](https://pages.github.com/) — hosting
